Abstract

What This Project Is About

Bridging the gap between clinical assessment and continuous, home-based joint monitoring.

Arthritis progressively limits joint mobility, grip strength, and overall hand function — yet most monitoring solutions require repeated clinic visits and costly equipment. This project introduces a Smart Arthritis Monitoring Sleeve that integrates wearable sensors with a web-based rehabilitation platform for continuous, objective joint assessment.

The sleeve captures three clinically relevant parameters: range of motion (wrist flexion, extension, and deviation), grip strength via force sensing, and self-reported swelling level. Sensor data streams in real time to a web dashboard where a Random Forest classifier predicts arthritis severity across four levels — Normal, Mild, Moderate, and Severe. Based on the predicted severity, the system recommends targeted rehabilitation exercises, monitored in real time using MediaPipe-based computer vision to ensure correct execution and count valid repetitions.
Objectives

Project Goals

To develop a portable, low-cost wearable system for continuous upper limb joint monitoring integrated with an intelligent rehabilitation platform.

Range of Motion Sensing
Measure wrist flexion, extension, and radial/ulnar deviation using an IMU sensor embedded in the sleeve.
Grip Strength Measurement
Quantify grip force using an FSR sensor with a calibrated measurement protocol.

Web Dashboard & Reports

CV Exercise Guidance

Home & Clinic Ready

System Workflow

How It Works — Step by Step

A structured protocol ensures consistent, clinically meaningful measurements every session.

1
Wear Device
Fit sleeve on wrist; enter patient details in web app to start session.
2
Baseline Calibration
Hold hand flat and still — IMU captures neutral wrist angle reference.
3
ROM Movements
Perform flexion, extension, radial & ulnar deviation as instructed.
4
ROM Calculated
Peak angles recorded, normalized against clinical reference values.
5
Grip Calibration
1-min baseline rest on FSR, then 10-sec maximum grip press.
6
Grip Scored
Force normalized against baseline to produce a percentage grip score.
7
Swelling Input
Patient self-reports swelling level via web interface.
8
Severity Predicted
Random Forest combines all inputs — outputs severity + confidence score.
ROM Protocol
  • Hold wrist flat — capture neutral reference
  • Flexion: bend wrist downward as far as comfortable
  • Extension: bend wrist upward as far as comfortable
  • Radial deviation: tilt toward thumb side
  • Ulnar deviation: tilt toward pinky side
  • % ROM calculated against clinical normal values
Grip Strength Protocol
  • Place palm on FSR pad for 1 minute (baseline)
  • System captures resting baseline force value
  • Squeeze FSR maximally for 10 seconds
  • Peak grip force recorded and normalized
  • Compared to clinical grip thresholds (N)
  • Combined with ROM + swelling for ML input

Hardware Sensing

ESP32 reads angular motion from the MPU6050 IMU over I2C and grip force from an FSR through a voltage divider. All readings stream continuously over Wi-Fi to the web platform.

Random Forest Classifier

Trained on ~6,000 clinically derived samples. Takes ROM %, normalized grip strength, and swelling score as inputs. Outputs one of four severity classes with confidence percentage.

Web Dashboard

Live sensor graphs, session logs, ML severity display with confidence, swelling self-assessment, and downloadable PDF patient diagnostic reports.

MediaPipe CV Tracking

Detects 21 hand landmarks per frame. Computes joint angles in real time. Valid repetitions counted only when movement matches expected biomechanical patterns.

Discussion

Analysis of Results

The system effectively demonstrated real-time joint monitoring and guided rehabilitation. The Random Forest Classifier achieved 99% accuracy on the held-out test set, with precision, recall, and F1-score all above 0.98 across all severity classes. The confusion matrix confirmed minimal misclassification — primarily between adjacent classes (e.g., Mild and Moderate), which is clinically acceptable.

Grip strength measurements correlated inversely with severity. Range of motion data captured meaningful variation across subjects, with flexion and extension angles proving most discriminative. The MediaPipe CV module reliably detected hand landmarks under indoor lighting, counting only biomechanically valid repetitions. Subjects completed exercise sessions with minimal coaching, indicating strong usability.

The web dashboard unified monitoring, visualization, and rehabilitation into a single accessible interface. Auto-generated PDF reports provide a practical deliverable for clinical handoff or personal tracking.
Conclusion

Summary

This project demonstrates that wearable sensing, machine learning, and computer vision can be meaningfully combined into a single, affordable arthritis monitoring and rehabilitation system. The sleeve reliably captures range of motion and grip strength; the Random Forest Classifier predicts severity with clinical-grade accuracy; and MediaPipe-powered exercise tracking ensures correct rehabilitation without specialist supervision.

The system is accessible, portable, and requires no complex clinical infrastructure — making it suitable for home use, remote monitoring, or low-resource clinical settings. Future work will focus on expanding the validated subject pool, incorporating clinical labels from rheumatologists, and refining the device form factor for long-term wearability.
